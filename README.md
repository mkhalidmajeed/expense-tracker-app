### Backend Setup
1. Navigate to `/backend`
2. Run `composer install`
3. Run `npm install`
4. Copy `.env.example` to `.env` and configure database
5. Run `php artisan migrate --seed`
6. Start server: `php artisan serve`

### Mobile Setup
1. Navigate to `/mobile`
2. Run `npm install`
3. Start app: `npx expo start`
